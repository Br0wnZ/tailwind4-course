# Tailwind CSS 4 Project

## Overview
This project demonstrates the latest features and capabilities of Tailwind CSS 4, showcasing modern web design and utility-first styling.

## Prerequisites
- Node.js (v18 or later)
- npm or yarn

## Setup and Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Br0wnZ/tailwind4-course.g sit
   cd tailwind4-course
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

## Project Structure
- `index.html`: Main project entry point
- `tailwind.config.js`: Tailwind CSS configuration
- `assets/`: Additional source files and styles

## Key Features
- Tailwind CSS 4 with latest utility classes
- Responsive design
- Customizable configuration
- Modern web development workflow

## Customization
Modify `tailwind.config.js` to customize:
- Color palette
- Breakpoints
- Custom utility classes
